# Red-Wine-criterion

## Sections

| Tab | What it does |
|---|---|
|  Case Study | Business framing, feature glossary, why quality prediction is tricky |
|  EDA & Visualization | Distributions, boxplots, correlation heatmap, scatter explorer |
|  Machine Learning | Train regression or classification models, view metrics, feature importance |
|  Prediction Playground | Slide in a wine's chemistry, get a predicted quality score/class |
|  3D Explorer | Interactive 3D scatter across any 3 features, colored by quality |
|  Raw Data & Export | Browse filtered data, download CSVs and the trained model (.pkl) |

The sidebar's quality-range filter applies to the EDA, 3D Explorer, and Raw Data tabs.
Model training always uses the full dataset. The model trained in the ML tab becomes
the active model in the Prediction Playground.
